Project List
LiteHttp
A simple, intelligent and flexible HTTP client for Android. With LiteHttp you can make HTTP request with only one line of code! It supports GET, POST, PUT, DELETE, HEAD, TRACE, OPTIONS and PATCH request types. LiteHttp could convert a java model to the parameter of http request and rander the response JSON as a java model intelligently. And you can extend the abstract class DataParser to parse inputstream to which you want.
Crossbow
An ameliorative, enhanced AsyncTask for Android. Crossbow also provides SimpleTask, SafeTask and CachedTask for rapid development. More convenient is, it has a TaskExecutor which can executes ordered, cyclicbarrier, delayed and timer Task.
Archive(on the way)
Archive is a fast, small, powerful ORM framework for Android. Archive makes you do CRUD operarions on SQLite database and keep your object relation mapping with a sigle line of code efficiently.
Tidy
A write less, do more, dependency injector for Android. Tidy makes your code more concise and development simpler by using annotation-based injection.
