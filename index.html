LiteHttp
A simple, flexible and intelligent HTTP client for Android. LiteHttp Supports GET, POST, PUT, DELETE, HEAD, TRACE, OPTIONS and PATCH request types. It could rander a simple java model as parameter when sent http request, and could rander the response JSON as a java model intelligently. And you can set a custom DataParser to parse inputstream to anything you want. It is easy to use and extend.
LiteAsync
An ameliorative, enhanced AsyncTask for Android. LiteAsync provides SimpleTask, SafeTask and CachedTask for rapid development. It also has a TaskExcutor to excute ordered, cyclicbarrier, delayed and timer Task.
LiteOrm
A fast, small, powerful ORM framework for Android. LiteOrm makes you handle CRUD operations of database on android platform just with a single line of code.
LiteIoc
A write less, do more, dependency injector for Android. LiteIoc makes code more concise and development simpler by usesing annotation-based injection.
